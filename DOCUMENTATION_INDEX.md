## 💡 Pro Tips

1. **Use VS Code**
   - Install Flutter extension
   - Use hot reload: Press `r` in terminal
   - Use hot restart: Press `R` in terminal

2. **Development**
   - Check `flutter analyze` for issues
   - Use `flutter doctor` for diagnostics
   - Check `.env` file for API configuration

   - Use breakpoints in VS Code
